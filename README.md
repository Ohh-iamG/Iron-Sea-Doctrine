## The Core Principle

*Iron — human industrial agency — may never permanently exceed the regenerative capacity of the systems it operates within.*

The Iron and Sea Doctrine states that every significant industrial action must record not only what it takes, but what it returns. It operationalises this through three rules:

1. **The Ledger Obligation** — physical return accounting, satellite-verifiable, linked to Return Metrics™
2. **The Sovereignty Check** — ecosystem-scale assessment before industrial action, linked to the 100-Domain Alignment Matrix™
3. **The Override Principle** — irreversible ecological damage triggers modification or cessation, enforced through the Ethics Gate™
